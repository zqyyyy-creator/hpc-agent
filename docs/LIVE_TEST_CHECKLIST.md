# HPC Agent Live Test Checklist

本文档用于在真实超算环境中验收 HPC Agent。它不会被程序自动执行，也不影响任何代码逻辑；作用是帮助开发者在演示前、阶段发布前，系统确认真实 HPC / Slurm / VASP 链路是否可用。

建议流程：

1. 先运行本地自动化检查：`.venv/bin/python tests/run_all_checks.py`
2. 确认本地检查通过后，再按本文档做真实环境测试。
3. 每次测试记录日期、模型、集群、Job ID、结果和问题。

注意：

- Live test 会连接真实超算，可能提交真实 Slurm 作业。
- VASP 测试会使用真实 VASP 输入文件和远端队列资源。
- 默认只测试短小作业，避免浪费队列资源。
- 涉及清理操作时，只在明确知道目标目录后执行。
